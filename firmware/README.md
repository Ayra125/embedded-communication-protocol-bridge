# Firmware

STM32-based firmware for interfacing I2C, SPI, and UART peripherals.

## Structure
- `Core/` — Main application entry point and system config
- `Drivers/` — Low-level protocol drivers (I2C, SPI, UART)
- `Peripherals/` — Device-specific abstractions (sensors, display, memory)
